# 대시보드 주요 버튼 색상 디자인

## 목표

교사용 대시보드의 브라운 계열 주요 버튼을 기존 세이지·민트·피치 팔레트와 자연스럽게 어울리는 딥 세이지로 통일한다.

## 적용 범위

- 상단의 `새 보드` 버튼 배경색
- 전체 보드 카드의 `열기` 버튼 배경색

## 색상

- 기본 배경: `#245F50`
- 글자 및 아이콘: 흰색
- 기존 모서리, 크기, hover 투명도, 비활성 상태는 유지한다.

## 제외 범위

- 사이드바 선택 메뉴와 토글 등 기존 primary 색상은 변경하지 않는다.
- 최근 보드의 텍스트형 `보드 열기` 링크는 변경하지 않는다.
- 보드 화면 내부의 색상은 변경하지 않는다.

## 검증

- `새 보드`와 카드 `열기` 버튼이 동일한 딥 세이지를 사용해야 한다.
- 기존 대시보드 레이아웃 및 전체 테스트가 유지되어야 한다.
