# Teacher Dashboard Color Design

## Direction

Use a white page background and place warm color only on cards, status surfaces, accents, and primary actions. This keeps the dashboard light while making information groups feel friendlier and easier to distinguish.

## Palette

- Primary berry: `#B33962`
- Supporting mint: `#6B9B73`
- Supporting coral: `#E18156`
- Text: `#30262B`
- Secondary text: `#725F66`
- Soft rose: `#F6DEE6`
- Soft mint: `#E2F0E5`
- Soft peach: `#FCE4D5`
- Warm outline: `#E4CDD5`

## Application

Primary controls and selected tabs use berry. Recent-board cards alternate soft rose and soft mint. Full-board cards remain white and cycle berry, mint, and coral top accents. Creation, last activity, and note count use soft rose, mint, and peach respectively. Inputs and neutral cards use warm outlines instead of blue-gray surfaces.

## Accessibility

Body copy remains dark on light surfaces. White text is limited to the berry primary surface. Supporting colors are not the sole status signal; text labels remain present.
